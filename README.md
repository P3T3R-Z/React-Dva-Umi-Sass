## 这是一个完整的前后端项目，此为前端部分，后端部分由EGG和MySql实现

后端在仓库的另一个项目里面

## Installation

```bash
$ npm install
$ npm start
$ open http://localhost:5284/
```

## 功能介绍

- Use `登录注册.
- Use `用户管理：增删改查，修改权限.
- Use `操作日志记录.
- Use `消息推送功能.
- Use `图片上传功能.
- Use `国际化demo.
- Use `Ipc demo.
